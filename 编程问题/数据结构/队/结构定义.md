# 队的结构定义

## 链式队的结构

需要有结点结构`QNode`

```c
typedef struct QNode {
    Queue_ElemType data;
    struct QNode *next; // 指向下一个结点
} QNode, *QNodePtr;
```

定义队，包含头指针和尾指针

```c
typedef struct {
    QNodePtr front;
    QNodePtr rear;
} LinkQueue;
```

> 当`Q->front == Q->rear`时，表示队列为空

## 顺序循环队列的结构定义

```c
#define SQ_QUEUE_MAX_SIZE 10;

typedef int SQ_Queue_ElemType;

typedef struct {
    SQ_Queue_ElemType *base;
    int front;
    int rear;
} Sq_Queue;
```

> 当`(Q->rear + 1) % SQ_QUEUE_MAX_SIZE == Q->front`时，表示队列已满
