# mysql字符串操作函数

```mysql
LENGTH()  -- 计算字符长度
RTRIM() -- 去空格
LTRIM() -- 去左空格
TRIM()  -- 去两边的空格
UPPER() -- 字母大写
LOWER() -- 字符小写
CONCATE() -- 合并与拼接字符串
CONCAT_WS() -- 按照某种固定分隔符分割字符串
SUBSTRING_INDEX() -- 按照固定模式分割字符串
LOCATE() -- 定位特定字符位置
LEFT()/RIGHT()/MID(Start,Length)  -- 从特定位置截取字符串
SUBSTRING(string,startlength) -- 特定位置截取固定长度字符串
REPLACE() -- 字符串替换函数
```
